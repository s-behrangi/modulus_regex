Calculate modulus with regular expressions

For any divisor d, remainder r, and base b, it's possible to generate a regular expression that matches exactly all n satisfying:

n % d = r

where n is represented in the given base b.

The algorithm has exponential complexity and the resulting regex grows just as fast, rendering this method purely theoretical for divisors much larger ~30 or so, but I think the process is neat.

You try:


match n % = in base
d > 0 0 ≤ r < d 1 ≤ b ≤ 16
